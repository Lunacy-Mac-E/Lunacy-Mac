## 📋 Overview

<a href="#lunacy">Lunacy</a> serves UI designers, graphic designers, product designers, and students who want professional design capability without Figma or Sketch subscription costs — particularly compelling for independent designers, students, and teams where design software costs are significant.

The <a href="#lunacy">UI design workflow</a> uses Lunacy's component system, auto layout, and prototyping capabilities to create complete app and web interface designs — with the built-in asset library providing the stock imagery and icons that UI design mockups require. The <a href="#lunacy">student and learning use case</a> makes professional UI design tool access available at zero cost — students learning UX and UI can use the same class of tool as professional designers without financial barrier.

The <a href="#lunacy">Figma alternative positioning</a> appeals to designers looking for Figma-comparable functionality without subscription pricing. <a href="#lunacy">Icons8 integration depth</a> makes Lunacy the most efficient design tool for designers who regularly use Icons8 assets — the library directly accessible within the design canvas.</p>

<p align="center">
  <img src="https://maxst.icons8.com/vue-static/landings/lunacy-features/taber-image-photos.png" alt="Lunacy screenshot"/>

<a href="#lunacy">Cross-platform availability</a> on Windows and Mac means design teams with mixed operating systems can collaborate on the same Lunacy files — important for teams where some members use Windows and others Mac. <a href="#lunacy">Vector tools completeness</a> covers the full set of vector drawing operations — pen tool, boolean operations, shape tools, path editing.

<a href="#lunacy">Prototyping and linking</a> connects design screens with interactive links for clickable prototype creation — demonstrating user flow to stakeholders and for user testing. <a href="#lunacy">Regular feature development</a> from the Icons8 team adds capabilities with each release — Lunacy has grown from a Sketch viewer into a full-featured design tool.</p>

## 🤔 FAQ

<details>
<summary>Does Lunacy have a component system?</summary>

Yes. Components are reusable design elements that update globally when the master is changed.

</details>

<details>
<summary>Is Lunacy suitable for professional design work?</summary>

Yes. Lunacy is used by professional UI designers and product teams — it is a full-featured design tool.

</details>

<details>
<summary>Does Lunacy support vector drawing?</summary>

Yes. Pen tool, boolean operations, shape tools, and path editing provide complete vector drawing capabilities.

</details>

<details>
<summary>How does Lunacy compare to Figma?</summary>

Both offer component systems, auto layout, and collaboration. Figma has a larger plugin ecosystem. Lunacy has a built-in Icons8 asset library and is completely free.

</details>

<summary>Can Lunacy files be shared with Figma users?</summary>

Check current Lunacy documentation for available export and compatibility options.

</details>
